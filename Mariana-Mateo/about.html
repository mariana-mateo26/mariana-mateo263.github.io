HTML: (HyperText Markup Language - Lenguaje de marcado de hipertexto)

¿Qué es?

HTML es el lenguaje base para crear la estructura y contenido de las páginas web. No es un lenguaje de programación, sino un lenguaje de marcado que usa etiquetas para organizar y mostrar información.

¿Para qué sirve?

Define elementos como encabezados, párrafos, imágenes, enlaces, formularios, tablas, etc.

Ventajas:

  • Es fácil de aprender y utilizar.
  • Compatible con todos los navegadores web.
